# Exam-Tracker-For-CUET
A dark-themed exam score tracker to log correct, wrong, and skipped answers across 5 subjects in real time.
A sleek, dark-themed exam tracking web app built with pure HTML, CSS, and JavaScript. Track your performance across 5 subjects — English, Biology, GAT, Chemistry, and Physics.
Features
Log correct (+5), wrong (−1), and skipped (0) answers per question
Real-time score and accuracy tracking
Visual progress bars and answer log
Combined score overview across all subjects
Undo last answer and reset per subject or all at once
Keyboard shortcuts (1, 2, 3) for fast logging
Saves progress automatically via localStorage
How to use
Open the app
Navigate to a subject tab
After each question, tap Correct, Wrong, or Skip
View your combined score on the Overview tab
